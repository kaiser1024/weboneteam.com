Chromebook to Linux

The precise problem definition: can you modify a Chromebook to behave like a standard UEFI-based x86 PC in order to enable Linux as the OP?

Background

Challenges in replacing ChromeOS with Linux on Chromebook

  • Chromebox has no UEFI (approximate equivalent of UEFI on stock Chrome: coreboot + Depthcarge)
  • stock Chromebox has no legacy BIOS as per factory configuration

Key questions

Can you replace coreboot + Deepthcharge with a real UEFI?

Answer: Yes, it is possible to install EDK II (UEFI) as a coreboot payload
